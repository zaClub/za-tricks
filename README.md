## Guide (资源 & 工具书)

### JavaScript
|网站|描述|
|:---|:---|
|[MDN](https://developer.mozilla.org/en-US/docs/Web/JavaScript)|JavaScript 权威网站|
|[JavaScript高级程序设计（第3版）](https://book.douban.com/subject/10546125/)|JavaScript 红宝书，一定要过一遍|

|网站|描述|
